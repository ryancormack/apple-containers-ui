# Volumes & Networks

## Volumes

The Volumes view lists all container volumes on your system.

<!-- screenshot: volumes-list — the volumes list view -->

Each volume shows its name, driver, and source path. Click the **Inspect** button to view the full JSON details.

## Networks

The Networks view lists all container networks.

<!-- screenshot: networks-list — the networks list view -->

Each network shows its name, IPv4 subnet, and IPv6 subnet. Click the **Inspect** button to view the full JSON details.
